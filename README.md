## Simple Money Transfer Api
Build using
1. JAVA8
1. Maven
1. Dropwizard
1. Junit
1. Mockito
1. PowerMockito
1. Jackson
1. Gson
1. Apache HttpUtils and Apache CommonUtils

---

### Build 
``` mvn clean package ```

### Run 
``` ./server.sh ```

### Test
``` mvn clean test```
